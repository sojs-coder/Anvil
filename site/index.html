About

Anvil.js was built with one goal in mind- to simplify the game-building process from start to finish. Anvil wants games to be able to be written quickly and bug-free on the first run.

The Anvil team has written multiple example games, all made in ~100 lines of JS code to ensure the ease-of-use of the engine

We are also actively working on a web interface for making games visually, as well as hosting for multiplayer games.
